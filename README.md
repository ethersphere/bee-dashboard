## Install

```
$ npm install -g @ethersphere/bee-dashboard
$ bee-dashboard
```

## Development

```sh
git clone git@github.com:ethersphere/bee-dashboard.git

cd  bee-dashboard

npm ci
npm run build
npm run serve
```

You can now access Bee Dashboard on [http://localhost:8080/](http://localhost:8080/)
